### Project Overview
This is a fully functional web application built with **Python** and **Flask**. It allows users to track their income and expenses in real-time.

### Technical Features:
* **CRUD Operations:** Users can Create, Read, Update, and Delete transactions.
* **Database Integration:** Utilizes a structured backend to store and retrieve financial data securely.
* **Dynamic UI:** Built with clean HTML/CSS templates for a responsive user experience.
